## 开发环境配置

直接 clone 到本地仓库，执行 gradlew clean shadowJar。

你应该可以在 build/libs/ 下找到带 -all 的 jar 文件。

如果你遇到了任何编译错误，你或许可以询问任意 AI 助手作为帮助，如豆包、Deepseek、Gemini……
